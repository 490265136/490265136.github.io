跳一跳外挂实现

最近朋友圈我发了个外挂截图

很巧合的是我没用外挂一不小心没忍住，玩到了1000+，大家都以为我用了外挂。

里面有很多解释，小白看不懂，就照着安装就ok 0.0

emmm，其实我没用外挂，事先声明下现在腾讯爸爸很强，能通过某些方式知道你用了外挂，取消你的外挂所刷出来的成绩。但是也有某些人通过某些方式可以绕过，所以大家可以尝试尝试

此外挂纯当学习，for fun，不要沉迷使用外挂哈

实现原理

1.把跳一跳的画面通过 WebDriverAgent 投到电脑端

2.通过鼠标点击起跳和落地点来计算距离

3.通过得到的距离乘以不同设备的时间系数得到按压时间

4 .根据得到的按压时间调用 WebDriverAgent 函数模拟点击屏幕过程

本文方法仅适用 Mac+iOS，其他平台原理差不多。Android 可以用 abd 代替 WebDriverAgent,本人木有任何android的机器，也只有Mac，所以没法进行教程编写。。。

实现步骤

基本准备

xcode

这是mac下苹果旗下的开发者工具，可以进行IOS上面的app开发，大家可以直接在app store中搜索下载

homebrew

homebrew是一个mac的安装集合，很多开发者工具只要在其中的库中搜索进行下载，比如python等等。
大家可以在一开始的终端输入一下代码，进行下载。

/usr/bin/ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)”

期间会要求输入密码，为你计算机的开机密码

安装WebDriverAgent

准备工作

从Github上下载密码

在终端中执行
cd downloads这样就确认你下载的内容是在downloads(下载)

git clone https://github.com/facebook/WebDriverAgent

这样就成功将node.js在downloads中下载看到

使用Homebrew安装Carthage

brew install Carthage

它是用来解决 xcode project 依赖的，可以把它看作是 cocoapod.

安装最新版的npm
git clone https://github.com/490265136/wechatjump

然后打开下载中wechatjump-master的文件夹，点击node-v8.9.4.pkg进行下载。

npm 是用来构建 Inspector， Inspector 是一个 react 的应用，通过往 WebDriverAgent 发 HTTP 请求，返回页面的结构

执行bootscrip脚本

在WebDriverAgent这个文件夹中执行以下操作，具体需要先通过终端进入WebDriverAgent这文件夹，首先前面下载的包是在downloads下，所以输入以下代码就可以进入

cd WebDriverAgent-master

cd ..可以返回上一个目录，，ls显示当前文件夹所有文件，文件夹名在代码前面。

然后在终端执行以下代码

./Scripts/bootstrap.sh

它会用Carthage下载所有的依赖，用npm打包响应的js文件

设置证书

因为最终程序需要在真机上跑，所以需要证书签名

双击打开WebDriverAgent.xcodeproj

确保Xcode版本最新

点击左上角的Xcode->Preference

在accent中登录自己的icoud，正常情况下可以看到team中显示personal team。

如下图 Signing 块所示，设置 WebDriverAgentLib 的签名(Bundle Identifier)

如果是免费的个人证书，那么要根据上图，把 Identify 的 Bundle Identifier 改成自定义的内容 还要根据下图把 Product Bundle Identifier 这一项改成之前的自定义内容

同样的事在 WebDriverAgentRunner 上再做一遍。

然后将agent和Runner中sign类的team选择成自己，如下图

进行真机测试

手机通过usb连接电脑

如图，在正方形右边的选项中选择WebDriverAgent和自己的手机，然后在Project中点击test,进行真机测试.

若手机闪一下黑色，然后退出后界面出现以下app和xcode下控制台出现类似以下信息，则表示测试成功

端口转发

国产手机可能不能通过手机端的 IP 和端口访问，解决方案是把手机端口投射到 Mac 上

安装 libimobiledevice

brew install libimobiledevice

装完后执行

iproxy 8100 8100

这时再用浏览器访问 localhost:8100/status，返回一段代码就表示安装成功

inspector 的地址是 localhost:8100/inspector，inspector 是用来查看 UI 的图层，也就是可以看到你手机的画面

一切无误，至此 WDA 安装完毕

python依赖库

该外挂需要运行在python的环境下

Mac自带的环境只有python2.7

大家可以打开终端,输入 python，按下回车就可以查询到相关版本，比如图所示，便是2.7.10版本。

如果不是3.X.X

之前下载的wechatjump-mater中已经下载好了python-3.6.4，大家可以直接下载。

接着在终端上开始操作。

如果一直按教程来，应该现在处于WebDriverAgent-master文件夹下，于是我们需要cd ..->cd wechatjump-master,使到wechatjump该文件夹下

然后执行以下代码

pip3 install -r requirements.txt

手机打开到微信跳一跳

直接点到第一次准备起跳的画面

开始执行外挂程序

终端内运行

python3 wechat_jump_iOS_py3.py

一般报错会报找不到库的错误，报出缺什么库就用 pip3 安装什么库

运行成功后依次点击弹出的窗口中的起始位置和目标位置，会自动计算距离后起跳。根据起跳的精准情况更改 wechat_jump_iOS_py3.py 中的 time_coefficient 参数，直到获得最佳取值

现在调试代码中有iphone5s以及iphone7的参数，其他适配手机需要自己调整

至此外挂运行成功 Enjoy jumping!

防止外挂被清除

腾讯加入了反外挂的功能，使用外挂可能会被清空成绩，严重情况账号可能会被隔离出排行榜

听说有朋友被关小黑屋了，就是排行榜就他一个人的那种，笑死我了0.0

听过一下几点可以绕过外挂：

1.起步跳几步后再执行外挂

2.途中不要贪心一直连续击中正中心

3.中途关闭外挂，自己跳几下，再开启外挂

4.快达到心仪的分数后，再手动跳几下

5.不要贪心不足蛇吞象

最后

这个纯当好玩，腾讯爸爸，我依旧是好孩子。拒绝外挂，从我做起。

我的一千多分真的是自己玩的，各位兄dei。。
